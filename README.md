# Products App

This is a small full-stack demo project with a .NET Web API and a React (MUI) frontend.

## What it does

- Create products
- View all products
- Filter products by colour
- Uses JWT authentication

## Tech

- .NET 6+ Web API
- Entity Framework Core
- React (Vite)
- Material UI
- Axios

## How to run

### Backend

1. Open the API project
2. Run the app

Make sure it's running on:
https://localhost:7164/
http://localhost:5016/

### Frontend

```bash
cd products-ui
npm install
npm run dev
```
